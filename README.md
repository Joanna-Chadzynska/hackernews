## Getting Started


### Installation

1. Clone the repo

```sh
git clone https://github.com/Joanna-Chadzynska/hackernews
```

2. Go to hacker-stories folder

```sh
cd hacker-stories
```

3. Install NPM packages

```sh
npm install
```

4. Run app

```sh
npm start
```
